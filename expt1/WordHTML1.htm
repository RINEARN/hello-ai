English:

Hello! this is a HTML file saved by Microsoft Word 2021.

How to Save:
* Menu Bar > File > Save As > Specify the file name, and select the type “Web Page, Filtered”

Also, we have removed header comments in the saved HTML file manually. They are style information (e.g.: font settings), so they may be a waste of tokens when AIs read the content.

日本語:

こんにちは！ このファイルは、Microsoft Word 2021で出力したHTMLファイルです。

出力方法:

  * メニューバー > ファイル > 名前を付けて保存 > ファイル名を指定し、「ファイルの種類」項目で「Webページ（フィルター後）」を選ぶ

なお、出力したHTML内のヘッダ付近に並ぶコメントは削っています。それらはフォント設定などのスタイル情報なので、AIがコンテンツを読む際にトークン数を無駄に消費してしまうのを避けるためです。
